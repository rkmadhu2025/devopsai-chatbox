## Overview

This platform provides an intelligent conversational interface for DevOps operations. It uses a multi-agent architecture where each agent specializes in specific tools and domains.

### Key Features

- **30+ Specialized Agents**: Each agent is an expert in specific DevOps tools
- **Auto-Routing**: Automatically routes queries to the most appropriate agent
- **Web UI**: Modern, responsive chat interface with markdown support
- **Tool Integrations**: Direct API connections to Prometheus, Grafana, Kubernetes, etc.
- **Multi-Channel**: Email, Slack, Teams, PagerDuty notifications
- **Codebase Analysis**: Analyze repositories for patterns and issues

## Installation

```bash
# Clone the repository
git clone <repository-url>
cd ContextEngineering

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
# or
.\venv\Scripts\activate   # Windows

# Install dependencies
pip install -r requirements.txt

# Configure
cp config.example.json config.json
# Edit config.json with your OpenAI API key
```

## API Reference

### REST Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | Web UI |
| GET | `/api/health` | Health check |
| GET | `/api/agents` | List all agents |
| GET | `/api/agents/categories` | Agents by category |
| GET | `/api/agents/{id}` | Get agent details |
| POST | `/api/chat` | Send message |
| GET | `/api/conversation/{id}` | Get history |
| DELETE | `/api/conversation/{id}` | Clear history |

### WebSocket

```javascript
// Connect
ws = new WebSocket('ws://localhost:8000/ws/session123');

// Send message
ws.send(JSON.stringify({
  message: "Create a Dockerfile",
  force_agent: null  // or "docker"
}));

// Receive response
ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log(data.response);
};
```
